# 데이터 안내

## 원칙

이 저장소는 공개 포트폴리오용 버전이며, 원본 데이터는 포함하지 않습니다.

## 포함하지 않은 자료

- 고객 원천 데이터 CSV
- 대회 제공 ZIP 파일
- 원본 데이터 명세 XLSX
- 발제 기관 제공 자료와 부속 문서

## 제외 이유

- 대회 제공 자료의 재배포 권한이 불명확합니다.
- 금융 고객 데이터는 비식별 여부와 별개로 공개 저장소 업로드에 신중해야 합니다.
- 이 저장소의 목적은 데이터 공개가 아니라 분석 설계와 전략 해석 역량을 보여주는 데 있습니다.

## 대신 제공하는 것

원본 데이터 대신 아래 자료를 통해 프로젝트를 이해할 수 있도록 구성했습니다.

- 프로젝트 요약 문서
- 분석 방법 문서
- 결과 및 전략 문서
- 재현 가이드
- 핵심 노트북 3종

즉, 데이터 자체는 비공개지만 `무엇을 문제로 정의했고`, `어떻게 분석했으며`, `어떤 전략으로 연결했는지`는 충분히 확인할 수 있습니다.
