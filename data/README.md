# 데이터 안내

이 저장소는 포트폴리오 공개용으로 정리한 버전이며, 원본 데이터는 포함하지 않습니다.

## 포함하지 않은 자료

- 고객 원천 데이터 CSV
- 대회 제공 ZIP 파일
- 원본 데이터 명세 XLSX
- 타 기관 발제 자료 및 샘플 데이터

## 제외 이유

- 대회 제공 자료의 재배포 권한이 불명확함
- 고객 데이터 성격상 공개 저장소 업로드가 적절하지 않음
- 포트폴리오의 목적은 데이터 자체보다 분석 구조와 해석 역량을 보여주는 데 있음

## 대체 정보

이 저장소에는 데이터 대신 아래 정보를 제공합니다.

- 분석 방법론 문서
- 최종 제출 기준 노트북 3종
- 재현 가이드
- 결과 요약 문서

원본 데이터 없이도 프로젝트의 문제 정의, 분석 흐름, 전략 도출 방식은 충분히 이해할 수 있도록 구성했습니다.
