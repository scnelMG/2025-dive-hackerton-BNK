# 재현 가이드

## 실행 순서

노트북은 아래 순서로 보는 것을 권장합니다.

1. [../notebooks/1.전처리_및_군집화.ipynb](../notebooks/1.전처리_및_군집화.ipynb)
2. [../notebooks/2.비이자이익_분석.ipynb](../notebooks/2.비이자이익_분석.ipynb)
3. [../notebooks/3.이자이익_분석.ipynb](../notebooks/3.이자이익_분석.ipynb)

## 필요한 입력 데이터

원본 노트북은 대회 제공 데이터와 일부 파생 파일을 참조합니다. 이 저장소에는 해당 파일을 포함하지 않습니다.

예상 입력 예시는 다음과 같습니다.

- 고객 원천 데이터 CSV
- 데이터 명세 XLSX
- 전처리 이후 파생 CSV

현재 확인 가능한 원본 폴더 기준으로는 아래 자료가 존재했습니다.

- `hacktho_FF.csv`
- `hacktho_FF_final.zip`
- `데이터명세_최종.xlsx`

단, 포트폴리오 저장소에는 위 파일을 복제하지 않았습니다.

## 왜 데이터를 제외했는가

- 공개 권한이 불명확한 대회 제공 자료이기 때문
- 고객 데이터 성격상 재배포를 피하는 편이 안전하기 때문
- 포트폴리오 목적상 분석 설계와 사고 과정을 보여주는 것이 우선이기 때문

## 실행 환경

기본 실행 환경은 다음과 같이 가정합니다.

- Python 3.9+
- Jupyter Notebook or JupyterLab
- pandas, numpy, matplotlib, seaborn, scikit-learn

설치는 루트의 [../requirements.txt](../requirements.txt)를 참고하면 됩니다.

## 주의 사항

- 일부 노트북은 해커톤 제출용 산출물이라 시각화 폰트 설정이나 경로 설정이 로컬 환경에 종속될 수 있습니다.
- 데이터 파일 경로는 로컬 환경에 맞게 수정해야 할 수 있습니다.
- 원본 데이터가 없으면 전체 재실행은 불가능하지만, 분석 흐름과 사용 방법은 코드 구조로 확인할 수 있습니다.
