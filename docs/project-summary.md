# 프로젝트 요약

## 한 줄 요약

BNK부산은행 고객 데이터를 이자이익과 비이자이익 관점으로 분리 해석해, 고객 세분화와 실행 전략을 함께 제안한 해커톤 프로젝트입니다.

## 프로젝트 배경

2025년 8월 DIVE 글로벌 해커톤에서 BNK부산은행이 제시한 과제는 다음과 같았습니다.

> 거래 유형별로 BNK 고객유형을 분류하고 분석하여 관리방안을 제언하라.

이 과제의 본질은 단순 분류가 아니라, 은행의 수익 구조 문제를 데이터로 해석하는 일이었습니다.  
팀은 프로젝트를 시작하며 다음 문제의식을 공유했습니다.

- 이자이익 중심 구조는 안정적이지만 외부 환경 변화에 취약할 수 있다.
- 비이자이익 상품 미이용 고객이 많다면 성장 전략의 여지가 크다.
- 고객을 하나의 기준으로만 분류하면 실제 관리 전략으로 연결하기 어렵다.

## 접근 방식 요약

본 프로젝트의 핵심 프레임은 **Two-track 분석**입니다.

- 이자이익 트랙: 안정적 수익 기반을 담당하는 고객과 상품 구조 분석
- 비이자이익 트랙: 성장성 있는 고객군과 교차 확장 가능성 분석

즉, 고객을 하나의 점수로 줄 세우는 대신 `어떤 수익 구조에 기여하는가`를 기준으로 해석했습니다.

## 핵심 결과

### 1. 비이자이익 성장 로드맵 도출

비이자이익 관점에서 고객을 `Basic → Standard → Premium → VIP`로 구분하고, 상위 군집으로 이동할수록 상품 보유와 거래 심화가 함께 커지는 패턴을 확인했습니다.

### 2. 교차 확장 기회 식별

카드, 예금, 펀드, 방카슈랑스 간 연관 패턴을 통해 단일 상품 판매보다 넓은 거래 확장을 유도하는 전략이 더 적합하다고 판단했습니다.

### 3. 이자이익 상품별 타깃 정리

대출 상품별로 고객 특성이 다르게 나타남을 확인하고, 상품군별 잠재고객 정의와 생애주기 관리 방향을 제안했습니다.
