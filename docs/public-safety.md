# 공개 안전성 검토

## Drive 매칭 결과

Google Drive에서 다음 자료를 확인했습니다.

- `data/`, `archive/`, `notebooks/`, `docs/`
- `.git` 백업 폴더
- 최종 발표자료와 보고서
- 제출 코드 ZIP
- BNK 관련 데이터/시각화 파일

## 공개 기준

이 저장소에는 포트폴리오 설명에 필요한 문서와 발표 산출물만 유지합니다. 원본 데이터, 발제 자료, 제출 코드 ZIP, Drive의 `.git` 백업은 추가 업로드하지 않습니다.

## 리뷰어 안내

이 프로젝트는 완전 재현형 저장소가 아니라, 은행 고객 세분화 문제를 어떻게 정의하고 분석 전략으로 전환했는지 보여주는 포트폴리오형 저장소입니다.
