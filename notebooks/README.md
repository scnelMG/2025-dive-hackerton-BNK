# 노트북 안내

## 포함한 노트북

이 폴더에는 최종 분석 흐름을 이해하는 데 필요한 핵심 노트북 3개만 남겼습니다.

1. `1.전처리_및_군집화.ipynb`
2. `2.비이자이익_분석.ipynb`
3. `3.이자이익_분석.ipynb`

## 읽는 순서

### 1. 전처리 및 군집화

- 구간형 데이터 정리
- 대표값 치환
- 정합성 점검
- 군집화 준비

### 2. 비이자이익 분석

- 비이자상품 중심 군집화
- 고객 성장 로드맵 해석
- 교차 확장 전략 도출

### 3. 이자이익 분석

- 대출 상품 중심 고객 프로필 분석
- 상품별 잠재고객 해석
- 관리 전략 제안

## 왜 이 3개만 남겼는가

포트폴리오 관점에서 중요한 것은 실험 파일 수가 아니라, 문제 정의부터 전략 제안까지 이어지는 핵심 흐름입니다.

따라서 이 저장소는 아래 내용을 가장 잘 보여주는 노트북만 유지했습니다.

- 전처리와 데이터 해석 방식
- Two-track 분석 구조
- 결과의 전략화 과정
