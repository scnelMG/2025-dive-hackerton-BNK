# Archive 안내

이 폴더는 **공개 저장소에 포함하지 않은 자료의 범위와 기준**을 정리한 공간입니다.

## 한눈에 보기

| 구분 | 포함 여부 | 설명 |
| --- | --- | --- |
| 대회 발제 자료 | 제외 | 제공 기관 자료로 재배포 범위가 불명확함 |
| 제출 원본 문서 | 제외 | 포트폴리오용 문서와 내용이 중복됨 |
| 중간 실험 노트북 | 제외 | 최종 분석 흐름과 직접 연결되지 않음 |
| 대용량 원본 파일 | 제외 | 저장소 관리와 공개 목적에 비효율적임 |

## 공개 저장소에서 제외한 항목

- `발제사` 폴더 전체
- `제출파일` 원본 문서 전체
- 중간 실험 노트북 및 중복 노트북
- 대용량 압축 파일 및 원본 데이터 파일

## 제외 기준

### 1. 공개 권한이 불명확한 자료

- 대회 제공 원본 자료
- 외부 기관 문서 또는 배포 범위가 명시되지 않은 파일

### 2. 포트폴리오 관점에서 중복되는 자료

- 최종 문서와 내용이 겹치는 제출용 원본
- 실험 과정만 남아 있는 중간 산출물

### 3. 저장소 운영에 비효율적인 자료

- 용량이 큰 압축 파일
- 원본 데이터 파일 전체

## 관리 원칙

- 원본 자료는 로컬 작업 디렉터리의 보관본으로 유지합니다.
- GitHub에는 분석 흐름을 설명하는 핵심 포트폴리오 자산만 포함합니다.
- 공개 저장소는 `재현 가능한 구조`와 `핵심 결과 이해`를 우선으로 구성합니다.
