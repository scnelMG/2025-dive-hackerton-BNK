# Archive 안내

이 폴더는 공개 저장소에서 제외한 자료의 범주를 설명하기 위한 자리입니다.

## 공개 저장소에서 제외한 항목

- `발제사` 폴더 전체
- `제출파일` 원본 문서 전체
- 중간 실험 노트북과 중복 노트북
- 대용량 압축 파일 및 원본 데이터 파일

## 이유

- 참고자료 성격이 강하거나 중복 산출물임
- 재배포 권한이 불명확함
- 포트폴리오 핵심 흐름을 흐릴 가능성이 있음

원본 자료는 현재 작업 디렉터리의 정리 전 보관본에서 유지하고, GitHub에는 선별된 포트폴리오 자산만 올리는 것을 전제로 합니다.
